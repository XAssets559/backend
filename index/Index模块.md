# Index模块

这是主页。需要以下几个部分：

- 文章推荐：5篇

- 任务列表

- 排行榜-->grades
- 广告（采用轮播图）


### 接口
server = 127.0.0.1:8000

#### 任务列表

url = 127.0.0.1:8000/task/

- 语法：get task/

- 请求参数

  - | 参数名 | 必选 | 说明                                                         |
    | ------ | ---- | ------------------------------------------------------------ |
    | change | 是   | 如果是编辑页面(教师后台中)，change = True,如果是主页，为False |

- 响应

  ```json
  [
      {
          "title": "第一题",
          "descriptions": "1234324",

      }
  ]
  ```
